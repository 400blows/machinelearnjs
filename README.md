# Kalimdor.js

Kalimdor.js is a Machine Learning library written in Typescript. It solves Machine Learning problems
and teaches users how Machine Learning algorithms work.

# User Installation

Using yarn

```bash
$ yan add kalimdor
```

Using NPM
```bash
$ npm install --save kalimdor
```
